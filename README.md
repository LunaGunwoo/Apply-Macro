# To start with pip

Please copy the code below and paste it into the terminal.

```
pip install -r requirements.txt
```

***

### 터미널 2개 실행 필요.
실행 종료시키고 싶으면 `ctrl + C` 누르면 종료.

#### 1. `python popup_hacker.py`
터미널에서 실행시켜 놓으면 광운대 수강신청 pop up 창 자동으로 수 입력하고 닫음

#### 2. `python main.py`
실행시켜 놓으면 1,2, ... 수로 매핑 시켜좋을 수 있음
